# Telegram Anonymous Message Bot

Бот для отправки анонимных сообщений в Telegram.

## Возможности

- Создание уникальных ссылок для получения анонимных сообщений
- Случайная генерация кодов ссылок (невозможно определить создателя)
- Информация об отправителе видна только администраторам
- Полное логирование всех действий в консоли

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` и добавьте токен бота:
```
BOT_TOKEN=ваш_токен_бота
```

4. Запустите бота:
```bash
python bot.py
```

## Использование

1. Отправьте `/start` боту
2. Нажмите кнопку "Создать ссылку для анон сообщений"
3. Поделитесь ссылкой с другими
4. Получайте анонимные сообщения!
